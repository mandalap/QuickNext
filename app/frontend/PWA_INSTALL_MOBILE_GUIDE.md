# 📱 Panduan Instalasi PWA di HP (Mobile)

## ✅ Ya, Aplikasi Bisa Diinstall di HP!

QuickKasir sudah dikonfigurasi sebagai **Progressive Web App (PWA)** dan bisa diinstall di HP Android dan iOS.

---

## 📲 Cara Install di Android

### **Metode 1: Install Prompt Otomatis (Paling Mudah)**

1. **Buka aplikasi di browser Chrome** di HP Android
2. **Tunggu beberapa detik** - akan muncul banner "Install QuickKasir" di bagian bawah layar
3. **Klik tombol "Install"** pada banner
4. **Konfirmasi install** - klik "Install" di popup yang muncul
5. **Selesai!** Aplikasi akan terinstall dan muncul di home screen

### **Metode 2: Menu Browser (Jika Banner Tidak Muncul)**

1. **Buka aplikasi di browser Chrome** di HP Android
2. **Klik menu** (3 titik di kanan atas)
3. **Pilih "Install app"** atau **"Add to Home screen"**
4. **Konfirmasi install** - klik "Install"
5. **Selesai!** Aplikasi akan terinstall

### **Metode 3: Popup Browser (Chrome)**

1. **Buka aplikasi di browser Chrome** di HP Android
2. **Scroll ke bawah** - Chrome akan menampilkan popup "Add QuickKasir to Home screen?"
3. **Klik "Add"** atau **"Install"**
4. **Selesai!** Aplikasi akan terinstall

---

## 🍎 Cara Install di iOS (iPhone/iPad)

### **Metode 1: Share Button (Safari)**

1. **Buka aplikasi di browser Safari** di iPhone/iPad
2. **Klik tombol Share** (kotak dengan panah ke atas) di bagian bawah
3. **Scroll ke bawah** dan pilih **"Add to Home Screen"**
4. **Edit nama** (opsional) - default: "QuickKasir"
5. **Klik "Add"** di kanan atas
6. **Selesai!** Aplikasi akan muncul di home screen

### **Metode 2: Menu Browser (Safari)**

1. **Buka aplikasi di browser Safari** di iPhone/iPad
2. **Klik tombol Share** (kotak dengan panah ke atas)
3. **Pilih "Add to Home Screen"**
4. **Konfirmasi** - klik "Add"
5. **Selesai!** Aplikasi akan terinstall

## ⚠️ Troubleshooting

### **Banner Install Tidak Muncul?**

**Solusi:**

1. **Pastikan menggunakan HTTPS** - PWA hanya bisa diinstall via HTTPS
2. **Clear cache browser** - Settings > Apps > Chrome > Clear Cache
3. **Update browser** - Pastikan menggunakan versi terbaru
4. **Coba metode manual** - Gunakan menu browser (Metode 2)

### **Tombol "Install" Tidak Berfungsi?**

**Solusi:**

1. **Refresh halaman** - Tekan tombol refresh
2. **Tutup dan buka lagi** - Tutup tab dan buka ulang
3. **Restart browser** - Tutup semua tab dan buka lagi
4. **Coba metode manual** - Gunakan menu browser

### **Aplikasi Tidak Bisa Dibuka Setelah Install?**

**Solusi:**

1. **Uninstall dan install ulang** - Hapus dari home screen dan install lagi
2. **Clear cache** - Settings > Apps > QuickKasir > Clear Cache
3. **Cek koneksi internet** - Pastikan ada koneksi untuk pertama kali

### **iOS: Tombol "Add to Home Screen" Tidak Ada?**

**Solusi:**

1. **Pastikan menggunakan Safari** - PWA iOS hanya support Safari
2. **Cek iOS version** - Minimal iOS 11.3
3. **Coba refresh** - Refresh halaman dan coba lagi
