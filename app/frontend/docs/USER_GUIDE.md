# 📖 QuickKasir User Guide

## Selamat Datang di QuickKasir!

QuickKasir adalah sistem POS (Point of Sale) modern untuk mengelola bisnis restaurant dan retail Anda dengan mudah.

---

## 🚀 Getting Started

### 1. Registrasi & Login

**Registrasi:**
1. Buka aplikasi QuickKasir
2. Klik "Daftar" atau "Register"
3. Isi form:
   - Nama lengkap
   - Email
   - Nomor WhatsApp
   - Password
4. Verifikasi nomor WhatsApp dengan OTP
5. Klik "Daftar"

**Login:**
1. Buka aplikasi QuickKasir
2. Masukkan email dan password
3. Klik "Masuk"

---

### 2. Lengkapi Profil

Setelah registrasi, lengkapi profil Anda:

1. **Nama Lengkap** - Wajib
2. **Email** - Wajib
3. **Nomor WhatsApp** - Wajib (sudah terverifikasi)
4. **Alamat** - Wajib
5. **Avatar** - Opsional

**Catatan:** Profil harus lengkap sebelum membuat bisnis.

---

### 3. Buat Bisnis Pertama

1. Setelah login, Anda akan diarahkan ke halaman "Buat Bisnis"
2. Isi form:
   - Nama Bisnis (wajib)
   - Email Bisnis (opsional)
   - Nomor Telepon (opsional)
   - Alamat (opsional)
   - Jenis Bisnis (wajib)
   - NPWP (opsional)
   - Tarif Pajak (default: 10%)
3. Klik "Buat Bisnis & Mulai"
4. Outlet pertama akan dibuat otomatis

---

## 📱 Install sebagai Aplikasi (PWA)

QuickKasir bisa di-install sebagai aplikasi di smartphone atau desktop:

### Desktop (Chrome/Edge):
1. Buka QuickKasir di browser
2. Klik icon "Install" di address bar
   - Atau klik banner "Install QuickKasir" di pojok kanan bawah
3. Klik "Install" di popup
4. Aplikasi akan terinstall dan muncul di desktop/start menu

### Mobile (Android):
1. Buka QuickKasir di Chrome
2. Banner "Install QuickKasir" akan muncul
3. Klik "Install"
4. Aplikasi akan terinstall dan muncul di app drawer

### Mobile (iOS):
1. Buka QuickKasir di Safari
2. Tap icon "Share"
3. Pilih "Add to Home Screen"
4. Aplikasi akan muncul di home screen

**Keuntungan Install:**
- ✅ Akses lebih cepat
- ✅ Bekerja offline
- ✅ Notifikasi update
- ✅ Pengalaman seperti aplikasi native

## 💼 Fitur Utama

### 1. Dashboard

Dashboard menampilkan:
- **Statistik Penjualan** - Total penjualan hari ini, bulan ini
- **Grafik Penjualan** - Visualisasi penjualan
- **Produk Terlaris** - Produk yang paling banyak terjual
- **Transaksi Terbaru** - Daftar transaksi terakhir
- **Quick Actions** - Akses cepat ke fitur utama

---

### 2. POS (Point of Sale)

**Membuat Transaksi:**
1. Buka menu "Kasir" > "POS"
2. Pilih produk dari daftar
3. Atur jumlah
4. Pilih customer (opsional)
5. Terapkan diskon (opsional)
6. Klik "Bayar"
7. Pilih metode pembayaran
8. Proses pembayaran
9. Print receipt (opsional)

**Fitur POS:**
- ✅ Scan barcode produk
- ✅ Pencarian produk cepat
- ✅ Multiple payment methods
- ✅ Diskon & kupon
- ✅ Deferred payment (Bayar Nanti)
- ✅ **Offline Mode** - Bekerja tanpa internet

---

### 3. Manajemen Produk

**Menambah Produk:**
1. Buka menu "Produk"
2. Klik "Tambah Produk"
3. Isi form:
   - Nama Produk
   - SKU/Barcode
   - Kategori
   - Harga
   - Stok
   - Gambar (opsional)
4. Klik "Simpan"

**Mengelola Produk:**
- Edit produk
- Hapus produk
- Update stok
- Upload gambar
- Set kategori

---

### 4. Manajemen Karyawan

**Menambah Karyawan:**
1. Buka menu "Karyawan"
2. Klik "Tambah Karyawan"
3. Isi form:
   - Nama
   - Email
   - Nomor Telepon
   - Role (Kasir, Kitchen, Waiter, Admin)
   - Gaji
4. Klik "Simpan"

**Catatan:** Jumlah karyawan dibatasi sesuai paket subscription Anda.

---

### 5. Manajemen Outlet

**Menambah Outlet:**
1. Buka menu "Pengaturan" > "Bisnis & Outlet"
2. Klik "Tambah Outlet"
3. Isi form:
   - Nama Outlet
   - Alamat
   - Nomor Telepon
   - Koordinat GPS (opsional)
4. Klik "Simpan"

**Catatan:** Jumlah outlet dibatasi sesuai paket subscription Anda.

---

### 6. Laporan

**Jenis Laporan:**
- **Laporan Penjualan** - Penjualan harian, bulanan, tahunan
- **Laporan Keuangan** - Pendapatan, pengeluaran, profit
- **Laporan Produk** - Produk terlaris, stok rendah
- **Laporan Karyawan** - Absensi, performa

**Cara Mengakses:**
1. Buka menu "Laporan"
2. Pilih jenis laporan
3. Pilih periode (hari, minggu, bulan, tahun)
4. Klik "Generate"
5. Export ke PDF/Excel (opsional)

## 🔄 Mode Offline

QuickKasir bisa bekerja **tanpa internet**!

### Fitur Offline:
- ✅ Buat transaksi
- ✅ Lihat produk (dari cache)
- ✅ Lihat customer (dari cache)
- ✅ Data tersimpan lokal
- ✅ Auto-sync ketika online kembali

### Cara Kerja:
1. Saat **online**: Data di-cache untuk digunakan offline
2. Saat **offline**: 
   - Transaksi disimpan lokal
   - Data dari cache ditampilkan
   - Indicator "Offline" muncul
3. Saat **online kembali**:
   - Data otomatis di-sync
   - Notifikasi "Koneksi dipulihkan"
   - Transaksi pending terkirim ke server

### Sync Indicator:
- **Bottom-left corner** menampilkan status sync
- Klik untuk manual sync
- Menampilkan jumlah transaksi pending

## ❓ Troubleshooting

### Masalah Umum

**1. Tidak bisa login**
- Pastikan email dan password benar
- Cek koneksi internet
- Coba reset password

**2. Transaksi tidak tersimpan**
- Cek koneksi internet
- Jika offline, transaksi akan tersimpan lokal
- Cek Sync Indicator untuk status sync

**3. Data tidak update**
- Refresh halaman (F5)
- Cek koneksi internet
- Clear cache browser (jika perlu)

**4. Install prompt tidak muncul**
- Pastikan menggunakan HTTPS (production)
- Pastikan browser mendukung PWA
- Cek apakah sudah terinstall sebelumnya

**5. Offline mode tidak bekerja**
- Pastikan service worker ter-register
- Cek di DevTools > Application > Service Workers
- Reload halaman
