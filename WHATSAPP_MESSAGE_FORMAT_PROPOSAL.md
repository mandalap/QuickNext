# 📱 Proposal Format Pesan WhatsApp Receipt

## 🎯 Tujuan
Membuat pesan WhatsApp yang:
- ✅ Lebih ringkas dan mudah dibaca
- ✅ Lebih menarik dan persuasif
- ✅ Tidak terlalu panjang
- ✅ Tetap informatif

---

## 📋 Opsi Format

### **Opsi 1: Format Ringkas (RECOMMENDED)**
**Tanpa detail item, hanya ringkasan + link**

```
✨ *Terima Kasih!*

Terima kasih *{nama_pelanggan}* telah berbelanja di *{nama_bisnis}*! 🙏

━━━━━━━━━━━━━━━━━━━━━━━━

📋 *STRUK PEMBAYARAN*

🆔 Order: *{order_number}*
📅 {tanggal} {waktu}
📍 {outlet_name}

━━━━━━━━━━━━━━━━━━━━━━━━

💰 *RINGKASAN PEMBAYARAN*

Subtotal    : Rp {subtotal}
Pajak       : Rp {pajak}
━━━━━━━━━━━━━━━━━━━━━━━━
💵 *TOTAL* : Rp {total}
💳 Dibayar : Rp {dibayar}
🔄 Kembalian: Rp {kembalian}

💳 Metode: {metode_pembayaran}

━━━━━━━━━━━━━━━━━━━━━━━━

📄 *Detail lengkap pesanan:*
🔗 {link_receipt}

💡 Simpan link ini untuk melihat detail transaksi kapan saja

━━━━━━━━━━━━━━━━━━━━━━━━

✨ *Terima kasih atas kunjungan Anda!*
💬 Ada pertanyaan? Balas pesan ini atau hubungi kami.

📞 {nomor_telepon}
```

**Keuntungan:**
- ✅ Sangat ringkas (sekitar 15-20 baris)
- ✅ Fokus pada informasi penting
- ✅ Link untuk detail lengkap
- ✅ Lebih mudah dibaca di mobile

---

Pajak       : Rp {pajak}

💵 *TOTAL* : Rp {total}
💳 Dibayar : Rp {dibayar}
🔄 Kembalian: Rp {kembalian}

💳 Metode: {metode_pembayaran}

- ✅ Link untuk detail lengkap
- ✅ Balance antara ringkas dan informatif

---

### **Opsi 3: Format Lengkap (DETAIL)**
**Dengan detail item lengkap (format saat ini, tapi diperbaiki)**

```
✨ *Terima Kasih!*

Terima kasih *{nama_pelanggan}* telah berbelanja di *{nama_bisnis}*! 🙏

━━━━━━━━━━━━━━━━━━━━━━━━

📋 *STRUK PEMBAYARAN*

🆔 Order: *{order_number}*
📅 {tanggal} {waktu}
📍 {outlet_name}

━━━━━━━━━━━━━━━━━━━━━━━━

📦 *DETAIL PESANAN:*

{daftar_item_dengan_format_ringkas}

━━━━━━━━━━━━━━━━━━━━━━━━

💰 *RINGKASAN PEMBAYARAN*

Subtotal    : Rp {subtotal}
Pajak       : Rp {pajak}
━━━━━━━━━━━━━━━━━━━━━━━━
💵 *TOTAL* : Rp {total}
💳 Dibayar : Rp {dibayar}
🔄 Kembalian: Rp {kembalian}

💳 Metode: {metode_pembayaran}

━━━━━━━━━━━━━━━━━━━━━━━━

📄 *Kuitansi Online:*
🔗 {link_receipt}

━━━━━━━━━━━━━━━━━━━━━━━━

✨ *Terima kasih atas kunjungan Anda!*

📞 {nomor_telepon}
```

**Keuntungan:**
- ✅ Lengkap dengan detail item
- ✅ Tidak perlu buka link untuk lihat item
- ✅ Cocok untuk pelanggan yang suka detail

**Kekurangan:**
- ❌ Lebih panjang (bisa 30-50 baris)
- ❌ Bisa terlalu panjang untuk pesanan banyak item

---

## 🎨 Format Item (untuk Opsi 3)

### **Format Ringkas Item:**
```
• {nama_produk} ({qty}x) = Rp {subtotal}
```

**Contoh:**
```
• Nasi Goreng (4x) = Rp 56.000
• Jeruk Ice (2x) = Rp 26.000
```

### **Format Detail Item (saat ini):**
```
• Nama Produk
  4 x Rp 14.000
  = Rp 56.000
```

---

## 💡 Rekomendasi

**Saya rekomendasikan Opsi 1 (Format Ringkas)** karena:
1. ✅ Paling ringkas dan mudah dibaca
2. ✅ Fokus pada informasi penting (total, pembayaran)
3. ✅ Link untuk detail lengkap jika diperlukan
4. ✅ Lebih cepat dibaca di mobile
5. ✅ Tidak overwhelming untuk pelanggan

**Alternatif:** Opsi 2 (Format Hybrid) jika ingin menampilkan jumlah item tanpa detail lengkap.

---

## 🔧 Implementasi

Format bisa dibuat:
1. **Configurable** - Admin bisa pilih format (Ringkas/Hybrid/Lengkap)
2. **Auto-detect** - Otomatis pilih format berdasarkan jumlah item
   - ≤ 3 item: Format Lengkap
   - 4-10 item: Format Hybrid
   - > 10 item: Format Ringkas

---

## 📝 Catatan

- Format menggunakan emoji untuk visual appeal
- Menggunakan bold (*text*) untuk emphasis
- Separator (━━━━━━━━━━━━━━━━━━━━━━━━) untuk struktur
- Link selalu disertakan untuk detail lengkap
- Call-to-action yang jelas dan friendly
