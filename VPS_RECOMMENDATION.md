## 🚀 Quick Start dengan DigitalOcean (RECOMMENDED)

### **Step 1: Buat Akun**

1. Daftar di https://www.digitalocean.com
2. Verifikasi email
3. Dapatkan **$200 free credit** (60 hari)

### **Step 2: Buat Droplet**

1. Klik "Create" → "Droplets"
2. Pilih:
   - **Image:** Ubuntu 22.04 LTS
   - **Plan:** Basic - $12/bulan (2GB RAM) ⭐
   - **Region:** Singapore (dekat Indonesia)
   - **Authentication:** SSH keys (recommended) atau Password
3. Klik "Create Droplet"

### **Step 3: Setup Server**

```bash
# Login ke server
ssh root@your-server-ip

# Update system
apt update && apt upgrade -y

# Install LEMP stack
apt install nginx mysql-server php8.2-fpm php8.2-cli php8.2-mysql php8.2-mbstring php8.2-xml php8.2-curl php8.2-zip -y

# Install Composer
curl -sS https://getcomposer.org/installer | php
mv composer.phar /usr/local/bin/composer

# Install Node.js 18
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
apt install -y nodejs

# Setup Firewall
ufw allow OpenSSH
ufw allow 'Nginx Full'
ufw enable
```

### **Step 4: Deploy Aplikasi**

Ikuti panduan di `DEPLOYMENT_RECOMMENDATION.md`

## 💡 Tips Memilih VPS

### **1. Lokasi Server:**

- **Singapore** - Terdekat dengan Indonesia (low latency)
- **Jakarta** - Jika pakai provider Indonesia (IDCloudHost, Biznet)
- **US/EU** - Lebih murah tapi latency lebih tinggi

### **2. Spesifikasi Minimal:**

- **RAM:** Minimal 1GB (recommended 2GB)
- **CPU:** Minimal 1 vCPU
- **Storage:** Minimal 25GB SSD
- **Bandwidth:** Minimal 1TB/bulan

### **3. Fitur Penting:**

- ✅ **SSD Storage** - Lebih cepat dari HDD
- ✅ **Backup** - Otomatis backup (optional, biasanya +20% harga)
- ✅ **Monitoring** - CPU, RAM, Network monitoring
- ✅ **Firewall** - Built-in firewall
- ✅ **IPv6** - Support IPv6 (optional)

## 🔒 Security Tips

1. **Gunakan SSH Keys** - Jangan pakai password untuk SSH
2. **Setup Firewall** - Hanya buka port yang diperlukan (80, 443, 22)
3. **Update Regularly** - `apt update && apt upgrade` setiap bulan
4. **Backup Otomatis** - Setup backup harian (jika tersedia)
5. **SSL Certificate** - Wajib pakai HTTPS (Let's Encrypt FREE)
